# radar_missile_system
Designed and simulated a radar-guided missile system in MATLAB Simulink, showcasing real-time target tracking, guidance algorithms, and control system dynamics.


🚀 Radar Guided Missile System (Simulink Project)
📌 Overview

This project presents a Radar-Guided Missile System designed and simulated using MATLAB Simulink. The system demonstrates how a missile detects, tracks, and intercepts a moving target using radar principles and control system techniques.

The model integrates signal processing, control systems, and dynamic motion modeling to replicate real-world missile guidance behavior.

🎯 Objectives

Simulate radar-based target detection

Implement missile guidance logic

Track and intercept a moving target

Visualize system performance in real-time

Understand control system dynamics in defense applications

🧠 Key Concepts Used

Radar Detection & Tracking

Relative Motion Dynamics

Feedback Control Systems

Proportional Navigation Guidance

Signal Processing

Coordinate Transformation

PID / Adaptive Control (if implemented)

🏗️ System Architecture

The system consists of the following main components:

📡 Radar System

Detects the target

Measures distance, velocity, and angle

Continuously updates target position

🎯 Target Model

Simulates moving object

Can follow linear or nonlinear trajectory

Adjustable speed and direction

🚀 Missile Model

Simulates missile dynamics

Receives guidance commands

Adjusts trajectory accordingly

🧭 Guidance System

Implements tracking algorithm

Uses Proportional Navigation (PN) or similar logic

Generates control signals for interception

🎮 Control System

Ensures stability and accuracy

Uses feedback loops

Minimizes tracking error

🔄 Working Principle

Radar detects the target and calculates its position

Target movement is continuously updated

Missile receives real-time target data

Guidance system computes interception path

Control system adjusts missile trajectory

Missile attempts to intercept the target
